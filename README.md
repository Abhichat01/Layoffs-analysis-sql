# 📊 Layoffs Data Analysis (SQL Project)

## 🔍 Project Overview
This project focuses on analyzing global layoffs data using SQL.  
The goal is to clean raw data, perform exploratory data analysis (EDA), and extract meaningful business insights.

---

## 📁 Dataset
- Source: Raw layoffs dataset (CSV format)
- Contains information about:
  - Company
  - Industry
  - Location
  - Total layoffs
  - Date

---

## 🛠️ Tools Used
- SQL Server (SSMS)
- Excel (for initial review)
- GitHub (for version control)

---

## 🧹 Data Cleaning Steps
- Removed duplicates
- Handled NULL and blank values
- Standardized text fields
- Converted date formats
- Fixed inconsistent data

---

## 📊 Exploratory Data Analysis (EDA)
Performed analysis to answer key questions:

- Total layoffs by year
- Top companies with highest layoffs
- Layoffs by industry
- Monthly trends
- Rolling total of layoffs using window functions
- Ranking companies using RANK() / DENSE_RANK()

---

## 🔥 Key Insights
- Certain years show peak layoffs trends
- Few companies contribute to majority layoffs
- Tech industry dominates layoffs
- Seasonal patterns observed in layoffs

---

## 📌 SQL Concepts Used
- CTEs (Common Table Expressions)
- Window Functions (SUM, RANK, DENSE_RANK)
- GROUP BY & Aggregations
- CASE Statements
- Joins & Data Cleaning Techniques
